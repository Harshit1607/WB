# 🪐 GravityShop

GravityShop is a high-fidelity, premium e-commerce platform built with **Next.js 16**, **Tailwind CSS**, and **Zustand**. Designed with a "Sleek Midnight" aesthetic, it prioritizes visual excellence, smooth micro-interactions, and a seamless user experience.

![Project Preview](https://images.unsplash.com/photo-1557821552-17105176677c?auto=format&fit=crop&q=80&w=1200)

🔗 **[Live Demo](https://gravity-8kda.onrender.com)**

## ✨ Features

- **🚀 Performance First**: Optimized with Next.js 16 App Router for blazing-fast page loads.
- **💎 Premium Design**: A curated "Sleek Midnight" theme featuring glassmorphism, depth-based shadows, and a modern Inter typography system.
- **🛒 Dynamic Shopping Cart**: Persistent cart state managed with Zustand, featuring real-time quantity updates and local storage synchronization.
- **🔍 Advanced Discovery**: Multi-criteria filtering (Category, Brand, Price) with instant search and URL parameter persistence.
- **✨ Fluid Motion**: Staggered grid entrances and smooth component transitions powered by Framer Motion.
- **📱 Mobile-First**: Fully responsive architecture ensuring a premium experience on any device.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Feedback**: [React Hot Toast](https://react-hot-toast.com/)

## 🎨 Design Philosophy

GravityShop follows a **minimalist yet expressive** design language:
- **Glassmorphism**: Subtle background blurs and border highlights to create layering and depth.
- **Typography**: Focused on readability and hierarchy using the Inter font family.
- **Feedback**: Interactive elements use micro-animations and toast notifications to keep the user informed.

## 🚀 Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Open [http://localhost:3000](http://localhost:3000)** in your browser.
